technique

  • JavaScript
    ES6+ ES6, ES2020 문법을 이해하고, 이를 기반으로 코드를 구현 할줄 압니다. Vanilla Javascript 플러그인, 프레임워크 등을 사용하지 않고 순수한 자바스크립트로 DOM을 제어하고 코드를 구현 할수 있습니다.
  • HTML
    HTML5 웹표준, 웹접근성을 준수하여 마크업을 하고 검색엔진 최적화(SEO)를 고려하여 가독성과 효율성에 맞게 태그를 사용합니다.
  • CSS
    CSS3 애니메이션과 미디어쿼리를 사용해 반응형 웹사이트를 구현할수 있으며, 유지보수성에 중점을 둔 코드를 구현합니다.
  • SCSS
    SCSS mixin, variable을 사용해 스타일을 캡슐화 해 중복되는 코드를 재활용 할수 있도록 코드를 구현 할수 있습니다.
  • Vue.js
    Vue.js vue.js의 생성주기를 이용해 데이터를 동적으로 사용하고, 라우터를 활용해 SPA 웹페이지를 구현할수 있습니다.
  • React
    React 컴포넌트를 세분화해 복잡한 UI를 구현할수 있습니다. 또한 hook을 사용하여 데이터를 최적화 하는데 중점을 둔 코드를 구현합니다. Styled-component 스타일드 컴포넌트를 사용한 코드를 구현 할수 있습니다.
  • git
    Git/Github git을 이용해 버전관리와 협업을 할수 있습니다.
