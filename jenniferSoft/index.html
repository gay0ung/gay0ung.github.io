실시간 모니터링

제니퍼는 모든 트랜잭션이 서버에 들어오는 순간부터 처리되기까지의 전 과정을 실시간으로 모니터링합니다.

개별 트랜잭션 분석

제니퍼의 스마트 프로파일링과 자동스택트레이스 기능을 통해 누구나 쉽고 빠르게 프로파일링 데이터와 느린 트랜잭션 원인을 분석할 수 있습니다..

분석과 보고서

제니퍼는 성능관리에 필요한 전문적인 분석 기능을 제공합니다. 고객은 제니퍼에서 수집하는 모든 데이터를 원하는 대로 활용할 수 있습니다..

클라우드 환경 모니터링

제니퍼는 AWS, AZURE, Docker 및 Docker 기반 컨테이너인 쿠버네티스(Kubernetes), 오픈시프트(OpenShift) 에서도 충분한 확장성과 안정성이 검증된 제품입니다.

전 세계 1,373개 이상의 고객사가
제니퍼를 사용하고 있습니다.

고객사 리스트 다운로드

국내 APM 시장 1위

제니퍼는 국내 시장에서 2005년부터 시장 점유율 65%를 유지하고 있는 국내 APM 1위 제품입니다.

높은 재구매율

재구매율은 업계 최고 수준입니다. 한 번 구매한 고객은 증설이나 신규 구매 시에도 제니퍼를 다시 구매하고 있습니다.높은 재구매율은 고객 신뢰의 표시입니다.

1,373+ 누적 고객 수

제니퍼는 전세계 1,373 고객사가 사용 중인 검증된 소프트웨어입니다. 매년 80~100여 개의 신규 고객사가 늘어나고 있습니다.

지원 플랫폼

JAVA

제니퍼는 현재 웹 애플리케이션 서버(WAS)에서 일어나는 모든 트랜잭션이 웹 애플리케이션 서버에 들어오는 순간부터 처리되기까지 전 과정을 실시간으로 모니터링합니다. 문제 발생시 쉽게 원인을 분석할 수 있으며,이를 통해 개발자와 운영자 모두가 쉽고 빠르게 서비스의 변화를 감지하고 대응할 수 있습니다.

더보기
.NET

제니퍼 모든 트랜잭션과 인프라 전반에 대한 정확하고 심층적인 상세 정보를 통해 운영 중인 시스템을 최소한의 부하로 상시 모니터링할 수 있습니다. 특히 데이터베이스와의 상호 연동에 따른 성능 이슈를 적시에 감지하여 한번의 클릭으로 코드 라인까지 근본 원인을 파악하며 해결할 수 있는 효과적인 프로세스를 제공합니다.

더보기
PHP

제니퍼는 PHP 플랫폼으로 동작하는 웹 애플리케이션을 실시간 모니터링 할 수 있습니다. 이 인터페이스는 트랜잭션이 어디서 처리되지 못하고 대기하고 있는지, 어떤 사용자가 현재 응답지연을 경험하고 있는지, 어느 SQL 쿼리가 현시점에서 수행되고 있는지와 같은 트랜잭션 수행상태에 대한 정보를 실시간으로 보여줍니다.
